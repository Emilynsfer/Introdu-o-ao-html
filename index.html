História da Web

  • Primeira geração de computadores

Em 1944 tinhamos o que é considerado o primeiro computador do mundo, o Mark 1, usado para realizar cálculos complexos com grande precisão e de forma rápida.

Em 1946 surgiu o Colossus criado pelos Alemães para realizar criptoanálises. Nesse mesmo ano tivemos o surgimento do ENIAC, considerado mais moderno comparado ao Mark 1.

Curiosidade: nessa primeira geração o termo BUG começou a ser bastante usado. Esse termo era usado para referir-se aos insetos que entravam nos computadores e geravam problemas, porque naquela época os computadores eram máquinas grandes, lugares fáceis para a entreda de insetos. Por causa disso, dizia-se que havia um bug (inseto) no sistema.

  • Segunda geração dos computadores

Nesse ano nós tivemos a evolução das válvulas para os transitores, o que fez com que os compuadores diminuíssem de tamanho. Isso entre 1959-1965.

  • Terceira geração

Utilização de circuitos integrados, os famosos chips. O que diminui ainda mais o tamanho dos computadores, porém aumentou muito a sua performance e qualidade de processamento. Por volta dos anos 1965-1970.

  • Quarta geração

Em 1971 surgiram os primeiros PCs, um grande avanço para a tecnologia na época. Nesse mesmo período a Apple lançou o primeiro mouse.

  • Quinta geração

Atualmente, estamos vivendo a quinta geração, marcada pelas criptomoedas, metaverso e inteligência artificial.

Como surgiu a Internet?

Surgiu através de uma forte necessidade de comunicação. A guerra fria contribuiu muito para o surgimento dos primeiros computadores e o surgimento de novas tecnologias

Joseph Carl Robnett Licklider teorizou sobre uma rede onde todos poderiam se comunicar através dela.

Devido ao fato dos computadores e recursos tecnológicos da época serem usados mais para fins militares, Tim Berners-Lee, em 1989, viu a oportunidade de unir hipertexto com TCP/IP e criar a World Wide Web (WWW), o primeiro navegador. Em 6 de agosto de 1991, ele publicou a primeira página web.

Conceito de Client

Client é o lado que consome a informação de algum lugar. Servidor é o lado que serve informações para os clientes.

O front-end é responsável por trabalhar do lado do cliente, ou seja, é responsável por garantir a melhor experiência para os usuários, pensando em responsividade e compatibilidade com diferentes navegadores.

Aplicações web

Aplicaões web são soluções criadas que possuem a internet como meio de comunicação entre Client x Server não sendo necessária a sua instalação. Aplicativos instalados não são Aplicações web!

Conceito de Server

Tipos de servidores:

  • Arquivos
  • Segurança (firewall)
  • Streaming
  • E-mail
  • Web

Hardware X Software

Parte lógica é o software, programas instalados no computador para realizarem funções específicas. Hardware são partes físicas do computador, como a memória, disco rígido e etc.

Um servidor possui um sistema operacional dentro dele, e dentro desse sistema operacional tem um sistema web instalado. O sistema web é o responsável por devolver a requisição do usuário para o cliente em forma de site, imagens e etc.

Linguagens de Programação

Linguagens de programação são linguagens escritas e formais que especificam um conjunto de instruções e regras usadas para gerarem softwares.

Algumas linguagens server-side:

  • PHP
  • Java
  • JavaScript
  • C#

Linguagem Server-side e Client-side mais usada:

  • JavaScript

HTML não é uma linguagem de programação! É um arquivo de marcação.

Intrdução ao HTML na prática

A página HTML possui 3 elementos principais:

  • tag: html
  • tag: head, onde as informções carregam antes de mostrarem o conteúdo ao usuário.
  • tag: body, onde o conteúdo aparece para o usuário.

Algumas tags:

  • i - deixa em itálico.
  • Strong - deixa o elemento em negrito.
  • Input - tag que se fecha sozinha, muito usada em formulários.
  • Img - tag que se fecha sozinha, usada para adicionar imagens no site.

Atributos das tags:

  • Id - identificador de elemento.
  • Style - para colocar comando css direto no elemento.
  • class - muito utilizado pelo css.

Tags de textos:

  • h1 até h6 - usado para títulos e subtítulos.
  • p - parágrafos no geral.
  • bockqoute - usado para citações.
  • u - underline.
  • mark - marca texto.
  • sup - geralmente usado para expoentes.
